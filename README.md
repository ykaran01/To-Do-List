# To-Do List

A simple **To-Do List** web application built with **HTML, CSS, and JavaScript** that allows users to add, complete, and delete tasks.

---

## Features

- **Add tasks**: Users can enter a task and add it to the list.  
- **Mark as complete**: Tasks can be marked as done with a checkbox.  
- **Delete tasks**: Remove individual tasks from the list.  
- **Filter tasks**: View completed and uncompleted tasks separately.

---

## Technologies Used

- **HTML**: Structure of the app.  
- **CSS**: Styling and layout.  
- **JavaScript**: Interactivity (add, delete, complete tasks).
